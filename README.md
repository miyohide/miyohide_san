MiyohideSan is a bot that announces one-week before a start of yokohamarb by e-mail or Twitter.

## Installation

Add this line to your application's Gemfile:

    gem 'miyohide_san', github: '1syo/miyohide_san'

And then execute:

    $ bundle

## Usage

    $ rake newborn
