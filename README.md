# MiyohideSan

MiyohideSan is a bot that announces one-week before a start of yokohamarb by e-mail or Twitter.

## Installation

Add this line to your application's Gemfile:

    gem 'miyohide_san'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install miyohide_san
